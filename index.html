Welcome to GCM example.

This example include using of GCM for Android and AppEngine, you can find how to use the new Volley library from IO 2013, Facebook SDK and G+ login.

you will need to get the following libraries before you can use this example: 1)https://www.facebook.com/campaign/landing.php?campaign_id=282184128580929&placement=Android_SDK&url=https://developers.facebook.com/resources/facebook-android-sdk-3.0.1.zip 2)git clone https://android.googlesource.com/platform/frameworks/volley 3)https://github.com/ianhanniballake/ActionBarSherlock#__sid=0
